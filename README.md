# Python ile İki Sayının Toplamını Bulma

Bu proje, Python'a yeni başlayanlar için geliştirilmiş
basit bir konsol uygulamasıdır. Kullanıcıdan alınan iki
sayının toplamını hesaplar.

## Özellikler
- Kullanıcıdan iki adet sayı alır
- Girilen değerleri kontrol eder
- Hatalı girişlerde kullanıcıyı uyarır

## Kullanılan Teknolojiler
- Python 3

## Nasıl Çalıştırılır?
Terminal veya komut satırında:

```bash
python iki_sayi_toplam.py
